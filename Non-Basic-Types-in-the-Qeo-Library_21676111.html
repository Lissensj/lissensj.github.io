Qeo Native Documentation : Non-Basic Types in the Qeo Library

Created by lissensj on May 21, 2014

About the Qeo Library

The Qeo library contains a number of non-basic types that have been added to qeo and which you can use. The QDM for these types is in the qeo_types.xml file,, which is located in the following folder in the SDK:

root > qdms>qdm>catalog

Current types

Currently, the library contains the following types:

  • DeviceId
  • UUID

How to Use these Types

To use these types, you need to include the qeo-types.xml file in your QDM files:

(...)
<include file="qeo_types.xml"/>
(...)
