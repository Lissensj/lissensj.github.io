Created by lissensj on May 21, 2014

Location of the QDM Files in the SDK

For more information about the contents of the SDK, refer to the Qeo SDK Installation Guides.

All QDM related files are located in the qdms subfolder in the Qeo SDK. These files consist of:

  • The available catalog of QDM files 
  • QDM file for the delivered samples
  • The QDM Catalog file and xsd file

These QDM files are intended as examples to demonstrate this feature. It is the intention to develop a "dictionary of Qeo standard data models" and to make this dictionary available for developers (e.g. as library of standard data models on the Internet).

The structure of the qdms folder is as follows:

  • qdm: contains two subfolders:
    • catalog: contains all general Qeo QDMs 
    • samples: contains all QDMs used in the samples.
  • xsd: contains the catalog file (see below) and the xsd file for validation of the QDM files.
Icon

Only a limited number of QDM files are available in the current release. Their purpose is to acquaint you with the format and structure of a QDM. A full library of ready-made QDMs is outside the scope of the SDK.

Importing the Catalog File into Eclipse

 Make sure you have Eclipse XML editors and tools installed. To import the catalog file for QDMs in Eclipse:

  1. Import both the XSD and catalog file to your Eclipse workspace
  2. Click on File > Import > XML Catalog.

 

To validate a QDM, open the QDM, right-click in the editor and click on Validate.

The Qeo XSD

Your QDM must comply to a specific structure. This structure is enforced using a XML Schema Definition File or XSD. This XSD file is located in the following subfolder: SDK Root Folder > qdms > xsd.

Code Generator

Once you have completed your QDMs, you can convert them into code objects of the (supported) code of your choice.The Code Generator supports all supported development languages.  For more information, refer to Code Generator User Guide.
